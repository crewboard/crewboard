# How to Use Drag and Drop in Chat

## Quick Start Guide

### Method 1: Drag and Drop from Windows Explorer

1. **Open a chat room** in the application
2. **Open Windows Explorer** and navigate to your files
3. **Select one or more files** (images, videos, or audio)
4. **Drag the files** over the chat window
   - You'll see a blue overlay with "Drop files here" message
5. **Release the mouse** to drop the files
6. **Review the attached files** in the preview area above the message input
7. **Type a message** (optional)
8. **Click the send button** to upload

### Method 2: Manual File Selection

1. **Open a chat room** in the application
2. **Click the attachment icon** (ğŸ“) in the message input area
3. **Select files** from the file picker dialog
   - You can select multiple files by holding Ctrl
4. **Click Open** to attach the files
5. **Review the attached files** in the preview area
6. **Type a message** (optional)
7. **Click the send button** to upload

## Tips

- **Multiple files**: You can attach multiple files at once
- **Mixed types**: Mix images, videos, and audio in one message
- **Text + Files**: Add a text message along with your files
- **Quick send**: Press Enter to send (works with files too)
- **Cancel**: Remove files before sending if you change your mind

## Keyboard Shortcuts

- **Enter**: Send message (with or without files)
- **Ctrl + V**: Paste files from clipboard (future feature)
- **Esc**: Clear all attached files (future feature)

## Troubleshooting

### Files not attaching?
- Check if the file type is supported
- Ensure the file exists and is accessible
- Try using the manual file picker instead

### Drag and drop not working?
- Make sure you're dragging over the chat area
- Ensure a chat room is selected
- Try restarting the application

### Files not sending?
- Check your internet connection
- Verify the server is running
- Check file permissions

## Examples

### Sending a Screenshot
1. Take a screenshot (Win + Shift + S)
2. Save it as a PNG file
3. Drag the PNG into the chat
4. Click send

### Sharing a Video Clip
1. Locate your video file in Explorer
2. Drag it into the chat window
3. Add a description message
4. Click send

### Sending Multiple Photos
1. Select multiple images in Explorer (Ctrl + Click)
2. Drag them all at once into chat
3. Review the preview
4. Click send to share all at once
